# Security Policy

Please do not disclose suspected security issues in public issues.

For wrapper-level issues, report privately to the repository owner using GitHub
private vulnerability reporting if it is enabled.

For issues in the underlying ML-KEM implementation, follow the upstream
`mlkem-native` security guidance:

https://github.com/pq-code-package/mlkem-native/security

This package is a Swift wrapper around `mlkem-native`; it does not claim FIPS
validation.
